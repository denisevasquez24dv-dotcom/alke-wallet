Alke wallet 
Descripción:Alke Wallet es una aplicación Front End de billetera digital que permite a los usuarios gestionar sus activos financieros de manera segura y sencilla, los usuarios pueden iniciar seción, enviar dinero a contactos y consultar su historial de tansacciones.
Tecnologías utilizadas:
-HTML
-CSS 
-JavaScript
-Bootstrap 
-JQuery 

Funcionalidades Principales 
-Registro e inicio de Sesión con credenciales seguras.
-Visualización y gestión de saldo disponible.
-Realizar depositos y retiros simulados 
-Enviar dinero a contactos dentro de la aplicación.
-Visualizar historial completo de las transacciones.
-Diseño responsivo para adaptarse a diferentes dispositivos.

Estructura del Proyecto 
Alke Wallet:
-CSS/  Arcchivo de estilo CSS
-js/ Archivo JavaScript
-index.html
-login.html, Pagina de inicio de sesión  
-menu.html, Menu principal 
-deposit.html, Pagina para depositos
-sendmoney.html, Pagina para enviar dinero 
-transactiosn.html, Pagina de historial de transacciones 
-README.md
